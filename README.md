# Birthday Reminder

Gimtadienių priminimo programa su grafine vartotojo sąsaja, sukurta Python kalba.

## Reikalavimai
- Python 3.10+
- pip

## Įdiegimas
```bash
git clone https://github.com/The5Anta/birthday-reminder
cd birthday-reminder
pip install -r requirements.txt
```

## Paleidimas
```bash
python main.py
```
